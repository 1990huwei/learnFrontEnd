## 理论知识学习

### 1. 语义化标签的重要性

**什么是语义化？**
- 使用有意义的标签来描述内容
- 标签名称能够表达内容的含义和结构
- 不仅仅是为了样式，更是为了内容的意义

**语义化的好处：**
1. **SEO优化**：搜索引擎更好地理解网页内容
2. **可访问性**：屏幕阅读器能更好地解读页面
3. **代码可读性**：开发者更容易理解和维护代码
4. **结构清晰**：页面结构更加清晰和有逻辑

### 2. HTML5语义化标签详解

**页面结构标签：**

```html
<!-- 页面头部 -->
<header>
    <h1>网站标题</h1>
    <nav>
        <ul>
            <li><a href="#">首页</a></li>
            <li><a href="#">关于</a></li>
            <li><a href="#">联系</a></li>
        </ul>
    </nav>
</header>

<!-- 导航区域 -->
<nav>
    <a href="#section1">第一部分</a>
    <a href="#section2">第二部分</a>
</nav>

<!-- 主要内容 -->
<main>
    <!-- 文章内容 -->
    <article>
        <header>
            <h2>文章标题</h2>
            <p>发布时间：<time datetime="2024-01-15">2024年1月15日</time></p>
            <p>作者：<address>张三</address></p>
        </header>
        
        <section>
            <h3>第一章节</h3>
            <p>章节内容...</p>
        </section>
        
        <section>
            <h3>第二章节</h3>
            <p>章节内容...</p>
        </section>
        
        <footer>
            <p>标签：<mark>HTML5</mark>, <mark>语义化</mark></p>
        </footer>
    </article>
    
    <!-- 侧边栏 -->
    <aside>
        <h3>相关文章</h3>
        <ul>
            <li><a href="#">HTML基础教程</a></li>
            <li><a href="#">CSS入门指南</a></li>
        </ul>
    </aside>
</main>

<!-- 页面底部 -->
<footer>
    <p>&copy; 2024 我的网站</p>
    <address>
        联系我们：<a href="mailto:contact@example.com">contact@example.com</a>
    </address>
</footer>
```

**内容语义标签：**

```html
<!-- 文章/独立内容 -->
<article>
    <h2>独立的文章标题</h2>
    <p>文章内容...</p>
</article>

<!-- 内容区块 -->
<section>
    <h3>区块标题</h3>
    <p>区块内容...</p>
</section>

<!-- 图片和说明 -->
<figure>
    <img src="chart.png" alt="销售图表">
    <figcaption>2024年销售数据图表</figcaption>
</figure>

<!-- 详细信息 -->
<details>
    <summary>点击查看详细信息</summary>
    <p>这里是详细的说明内容...</p>
</details>

<!-- 时间标记 -->
<time datetime="2024-01-15T10:30:00">2024年1月15日上午10:30</time>

<!-- 进度条 -->
<progress value="70" max="100">70%</progress>

<!-- 度量值 -->
<meter value="6" min="0" max="10">6 out of 10</meter>
```

### 4. 表单的创建和使用

**基础表单结构：**

```html
<form action="/submit" method="post">
    <fieldset>
        <legend>个人信息</legend>
        
        <!-- 文本输入 -->
        <label for="name">姓名：</label>
        <input type="text" id="name" name="name" required>
        
        <!-- 邮箱输入 -->
        <label for="email">邮箱：</label>
        <input type="email" id="email" name="email" required>
        
        <!-- 密码输入 -->
        <label for="password">密码：</label>
        <input type="password" id="password" name="password" required>
        
        <!-- 数字输入 -->
        <label for="age">年龄：</label>
        <input type="number" id="age" name="age" min="1" max="120">
        
        <!-- 日期输入 -->
        <label for="birthday">生日：</label>
        <input type="date" id="birthday" name="birthday">
    </fieldset>
    
    <fieldset>
        <legend>其他信息</legend>
        
        <!-- 单选按钮 -->
        <p>性别：</p>
        <input type="radio" id="male" name="gender" value="male">
        <label for="male">男</label>
        
        <input type="radio" id="female" name="gender" value="female">
        <label for="female">女</label>
        
        <!-- 复选框 -->
        <p>兴趣爱好：</p>
        <input type="checkbox" id="reading" name="hobbies" value="reading">
        <label for="reading">阅读</label>
        
        <input type="checkbox" id="music" name="hobbies" value="music">
        <label for="music">音乐</label>
        
        <input type="checkbox" id="sports" name="hobbies" value="sports">
        <label for="sports">运动</label>
        
        <!-- 下拉选择 -->
        <label for="city">城市：</label>
        <select id="city" name="city">
            <option value="">请选择城市</option>
            <option value="beijing">北京</option>
            <option value="shanghai">上海</option>
            <option value="guangzhou">广州</option>
            <option value="shenzhen">深圳</option>
        </select>
        
        <!-- 多行文本 -->
        <label for="message">留言：</label>
        <textarea id="message" name="message" rows="4" cols="50" placeholder="请输入您的留言..."></textarea>
    </fieldset>
    
    <!-- 提交按钮 -->
    <button type="submit">提交</button>
    <button type="reset">重置</button>
</form>
```

**HTML5新增输入类型：**

```html
<!-- 搜索框 -->
<input type="search" placeholder="搜索...">

<!-- 电话号码 -->
<input type="tel" placeholder="请输入电话号码">

<!-- 网址 -->
<input type="url" placeholder="请输入网址">

<!-- 颜色选择器 -->
<input type="color" value="#ff0000">

<!-- 范围滑块 -->
<input type="range" min="0" max="100" value="50">

<!-- 时间输入 -->
<input type="time">

<!-- 日期时间输入 -->
<input type="datetime-local">

<!-- 月份输入 -->
<input type="month">

<!-- 周输入 -->
<input type="week">
```

### 5. HTML5多媒体标签

**音频标签：**

```html
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    您的浏览器不支持音频播放。
</audio>

<!-- 自动播放（谨慎使用） -->
<audio controls autoplay loop>
    <source src="background.mp3" type="audio/mpeg">
</audio>
```

**视频标签：**

```html
<video controls width="640" height="360">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    <track src="subtitles.vtt" kind="subtitles" srclang="zh" label="中文字幕">
    您的浏览器不支持视频播放。
</video>

<!-- 带海报图片的视频 -->
<video controls poster="video-poster.jpg">
    <source src="video.mp4" type="video/mp4">
</video>
```

### 常见问题解答

**Q: 什么时候使用article，什么时候使用section？**
A: article用于独立的、完整的内容（如文章、博客帖子）；section用于文档中的主题性分组。

**Q: 表单提交后没有反应？**
A: 检查form标签的action属性是否正确，method属性是否设置。在学习阶段可以暂时设置action="#"。

**Q: 表格太宽怎么办？**
A: 可以使用CSS控制表格宽度，或者考虑在移动设备上使用响应式设计。

**Q: 如何让表单更用户友好？**
A: 使用placeholder提示、required验证、合理的label标签，以及清晰的fieldset分组。
