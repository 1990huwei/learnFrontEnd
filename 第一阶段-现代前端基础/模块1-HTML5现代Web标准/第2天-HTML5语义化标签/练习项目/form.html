HTML5表单新特性总结

新增输入类型

  • email - 邮箱地址，自动验证格式
  • tel - 电话号码
  • url - 网址，自动验证URL格式
  • number - 数字，支持min、max、step属性
  • range - 滑块，用于选择数值范围
  • date - 日期选择器
  • datetime-local - 本地日期时间
  • time - 时间选择器
  • color - 颜色选择器
  • search - 搜索框
  • file - 文件上传，支持accept属性

表单验证属性

  • required - 必填字段
  • pattern - 正则表达式验证
  • minlength/maxlength - 字符长度限制
  • min/max - 数值范围限制
  • step - 数值步长
  • placeholder - 占位符文本

表单结构标签

  • <form> - 表单容器
  • <fieldset> - 表单字段分组
  • <legend> - 字段组标题
  • <label> - 标签，提升无障碍性
  • <input> - 输入控件
  • <textarea> - 多行文本
  • <select> - 下拉选择
  • <option> - 选项
  • <button> - 按钮
