表格使用技巧总结

表格结构标签

  • <table> - 表格容器
  • <caption> - 表格标题
  • <thead> - 表格头部
  • <tbody> - 表格主体
  • <tfoot> - 表格底部
  • <tr> - 表格行
  • <th> - 表头单元格
  • <td> - 数据单元格

单元格合并

  • colspan - 跨列合并
  • rowspan - 跨行合并

注意：合并单元格时要注意调整其他行的单元格数量。

表格无障碍性

  • 使用 <caption> 为表格提供描述
  • 使用 <th> 标记表头
  • 为复杂表格添加 scope 属性
  • 使用 summary 属性描述表格结构（HTML5中已废弃，可用aria-describedby替代）
