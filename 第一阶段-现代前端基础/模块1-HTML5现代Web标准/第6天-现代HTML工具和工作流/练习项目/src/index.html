开发工作流

从开发到部署的完整工作流程

1

项目初始化

使用npm初始化项目，安装必要的开发依赖和工具。

npm install
2

开发环境

启动Vite开发服务器，享受热模块替换和快速构建。

npm run dev
3

代码开发

编写HTML、CSS和JavaScript代码，工具会自动格式化和检查。

npm run format
4

质量检查

运行代码检查工具，确保代码质量和最佳实践。

npm run lint
5

生产构建

构建优化的生产版本，包括代码压缩和资源优化。

npm run build
6

预览部署

预览生产构建结果，确保一切正常后进行部署。

npm run preview
